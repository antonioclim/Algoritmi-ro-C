# Format: Prima linie conține numărul de noduri, a doua linie conține valorile.
# Arborele este construit ca arbore binar complet din array.

#
# Pentru nodul la index i:
#   - Copilul stâng este la index 2*i + 1
#   - Copilul drept este la index 2*i + 2
#
# =============================================================================

#
#            50

#         30    70

#       20  40 60  80
#
# Array: [50, 30, 70, 20, 40, 60, 80]

# Exemplul 2: Arbore incomplet cu noduri lipsă
#
#            10
#           /  \
#          5    15
#         /      \
#        3        20
#
# Array: [10, 5, 15, 3, -1, -1, 20]
# (-1 indică poziții goale)

# Exemplul 3: Arbore degenerat (linked list)
#
#     1
#      \
#       2
#        \
#         3
#          \
#           4
#
# Array: [1, -1, 2, -1, -1, -1, 3, ...]
# (Dificil de reprezentat în format array, mai bine construcție manuală)
